Project Overview

This project presents an interactive data analysis dashboard built using the Zomato restaurant dataset, aimed at uncovering insights into restaurant distribution, cuisine trends, pricing patterns, ratings, and service availability across multiple countries and cities. The dashboard is designed to support data-driven decision-making for stakeholders such as restaurant owners, food delivery platforms, and business analysts by visualizing key performance indicators (KPIs) and customer behavior patterns.
